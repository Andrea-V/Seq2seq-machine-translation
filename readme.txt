Questa cartella contiene i seguenti files:

- preprocessing.py: funzioni usate per il preprocessing del dataset.
- language_index.py: implementazione dell'indice e indice inverso a partire da un determinato vocabolario.
- encoder.py: implementazione dell'encoder del modello seq2seq.
- decoder.py: implementazione del decoder del modello seq2seq.
- main.py: script principale, crea il dataset ed esegue il training del modello.
- eval.py: funzioni per il test del modello e il plot degli attention weights.
- valenti_hlt_project.pdf: relazione del progetto.
- readme.txt: questo file. 

Per eseguire il progetto, lanciare lo script main.py mediante l'interprete python (versione 3.5).
